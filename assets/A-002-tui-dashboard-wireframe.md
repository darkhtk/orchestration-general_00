## 화면 구성 요소 상세

### 1. 헤더 바 (라인 1-2)
- **시간**: 현재 시각 및 자동 새로고침 간격
- **비용 추적**: 실시간 API 사용 비용 / 일일 예산
- **알림 카운터**: 현재 활성 알림 개수

### 2. 좌측 상단: 에이전트 상태 패널 (라인 3-14)
- **에이전트별 상태**: 🟢 ACTIVE / 🔴 ERROR / 🟡 PAUSED / ⚫ STOPPED
- **루프 카운터**: 현재 실행 횟수 / 최대 실행 횟수 (∞ = 무제한)
- **타이머 정보**: 현재 작업 경과 시간 / 다음 실행까지 남은 시간
- **최근 액션**: 마지막으로 완료한 작업 요약

### 3. 우측 상단: BOARD 요약 패널 (라인 3-14)
- **태스크 개요**: 상태별 태스크 개수 및 진행률
- **스프린트 정보**: 현재 스프린트 진행 상황
- **우선순위 큐**: P0/P1/P2 별 태스크 개수

### 4. 중앙: 최근 활동 로그 (라인 15-26)
- **실시간 피드**: 모든 에이전트의 최신 활동 10개 항목
- **타임스탬프**: 정확한 시각 정보
- **에이전트 아이콘**: 👔 Supervisor, 🤖 Developer, 🕵️ Client, 📅 Coordinator
- **액션 상태**: ✅ 완료, 🔨 진행중, 📝 리뷰, 🐛 버그수정, ⚠️ 경고

### 5. 좌측 하단: 알림 및 에러 패널 (라인 27-38)
- **심각도별 분류**: 🔴 Critical, 🟡 Warning, 🔵 Info
- **최근 항목**: 시간순 정렬된 알림 목록
- **필터링**: 심각도별 토글 표시/숨김

### 6. 우측 하단: 성능 지표 패널 (라인 27-38)
- **태스크 메트릭**: 평균 처리 시간, 시간당 완료 수
- **비용 효율**: 태스크당 API 비용
- **성공률**: 에러 없이 완료된 비율
- **일일 통계**: 오늘 완료된 작업 요약

### 7. 하단: 단축키 바 (라인 39)
- **키보드 단축키**: 주요 기능 빠른 접근
- **모달 창 호출**: 상세 정보 및 설정 화면

## 인터랙션 기능

### 키보드 단축키
- **R**: 강제 새로고침 (3초 대기 무시)
- **P**: 모든 에이전트 일시정지/재개
- **L**: 상세 로그 뷰어 모달 열기
- **C**: 설정 편집 모달 열기
- **T**: BOARD 상세 뷰어 모달 열기
- **A**: 에이전트별 상세 정보 모달 열기
- **Q**: 안전한 종료 (에이전트 정리 후 종료)

### 마우스 인터랙션 (선택사항)
- **패널 클릭**: 해당 영역 상세 보기 모달
- **에이전트 상태 클릭**: 에이전트별 제어 메뉴
- **태스크 클릭**: 태스크 상세 정보 및 편집

### 색상 스키마
- **상태 색상**: 🟢 정상, 🟡 주의, 🔴 에러, ⚫ 중지
- **우선순위**: P0=빨강, P1=주황, P2=노랑, P3=파랑
- **진행률**: 0-25%=빨강, 26-75%=노랑, 76-100%=초록

## 반응형 레이아웃

### 대형 터미널 (140x40+)
- 추가 패널: 커밋 히스토리, 에이전트 상세 로그
- 더 많은 최근 활동 표시 (15-20개 항목)

### 중형 터미널 (100x25)
- 패널 2x2 배치
- 최근 활동 축약 (5개 항목)
- 단축키만 표시

### 소형 터미널 (80x20)
- 단일 패널 탭 전환 방식
- 필수 정보만 표시
- 간소화된 인터페이스

## 데이터 업데이트 주기

- **실시간**: 에이전트 상태, 태스크 진행률
- **3초 간격**: 로그 피드, 성능 지표
- **10초 간격**: BOARD 요약, 알림 체크
- **60초 간격**: API 비용, 일일 통계
