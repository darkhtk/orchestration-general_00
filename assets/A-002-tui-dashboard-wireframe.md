## 컴포넌트 상세 설명

### 1. 헤더 (Header Bar)
```
위치: 상단 2줄
내용:
├─ 제목: "4-Agent Orchestration Dashboard"
├─ 전체 상태: RUNNING/PAUSED/ERROR
├─ 프로젝트명: project.config.md에서 읽기
├─ 현재시각: 실시간 업데이트
└─ 예산 현황: 사용량/총액 + 퍼센트 바
```

### 2. 에이전트 상태 패널 (Agent Status Panel)
```
위치: 3-6줄 (4줄)
구조: 테이블 형태
┌───────┬─────────┬────────────┬──────────────────┬────────────┬──────────────┐
│ Agent │ Status  │ Last Run   │ Current Step     │ Next Run   │ Stats        │
├───────┼─────────┼────────────┼──────────────────┼────────────┼──────────────┤
│ Coord │ ●ACTIVE │ HH:MM:SS   │ Current action   │ HH:MM:SS   │ ✓ok ✗err ↺retry │
│ Super │ ○IDLE   │ HH:MM:SS   │ -                │ trigger    │ ✓ok ✗err ↺retry │
│ ...   │         │            │                  │            │              │
└───────┴─────────┴────────────┴──────────────────┴────────────┴──────────────┘

상태 아이콘:
●ACTIVE   = 현재 실행중
○IDLE     = 대기중
⚠WARN     = 경고 상태
❌ERROR   = 에러 상태
🔄RESTART = 재시작중
```

### 3. BOARD 요약 패널 (Board Summary Panel)
```
위치: 7-16줄 (10줄)
구조:
├─ 우선순위별 카운트 (1줄)
├─ 공백 (1줄)
├─ 현재 진행중인 태스크 목록 (4줄)
├─ 공백 (1줄)
└─ 높은 우선순위 대기 태스크 목록 (3줄)

표시 방식:
🔥 P0 (Critical) - 빨간색
🟡 P1 (High)     - 노란색
🔵 P2 (Normal)   - 파란색
📋 Backlog       - 회색
⏳ In Progress  - 초록색
✅ Done(today)  - 밝은 초록색
```

### 4. 실시간 로그 패널 (Live Logs Panel)
```
위치: 17-27줄 (11줄)
기능:
├─ 자동 새로고침 (3초 간격)
├─ 최신 10개 항목 표시
├─ 에이전트별 색상 구분
├─ 타임스탬프 + 레벨 + 메시지
└─ 스크롤 가능 (PgUp/PgDown)

형식: [HH:MM:SS] [Agent] 아이콘 메시지
아이콘:
📧 - 메일/알림
🔍 - 모니터링
✅ - 완료
⚡ - 스케줄
📋 - 태스크 관리
💤 - 대기 상태
🔄 - 시작/재시작
❌ - 에러
🚀 - 할당
⏰ - 시간 기반 액션
```

### 5. 에러 모니터 패널 (Error Monitor Panel)
```
위치: 28-34줄 (7줄)
구조:
├─ 레벨별 카운트 (1줄)
├─ 공백 (1줄)
└─ 최근 에러 3건 (3줄)

레벨:
🔴 CRITICAL - 시스템 중단
🟠 ERROR    - 작업 실패
🟡 WARNING  - 주의 필요
ℹ️ INFO     - 일반 정보
```

### 6. 하단 명령어 바 (Command Bar)
```
위치: 하단 1줄
명령어:
[Q]uit     - 종료
[R]efresh  - 수동 새로고침
[F]ilter   - 로그 필터링
[D]etails  - 상세 정보 팝업
[H]elp     - 도움말

상태 정보:
CPU: 현재 CPU 사용률
MEM: 현재 메모리 사용량
```

## 키보드 단축키

### 전역 단축키
```
q/Q     : 종료
r/R     : 새로고침
f/F     : 로그 필터 토글
d/D     : 상세 팝업
h/H     : 도움말
ESC     : 현재 모드 취소
```

### 내비게이션
```
↑/↓     : 로그 스크롤
PgUp/Dn : 로그 페이지 스크롤
Tab     : 패널 간 이동
Enter   : 선택된 항목 상세보기
```

### 빠른 액션
```
1-4     : 에이전트 상세보기 (Coord=1, Super=2...)
p       : 일시정지/재개
s       : 스케줄러 강제 실행
c       : 로그 클리어
```

## 성능 최적화

### 업데이트 전략
```
실시간 섹션: 1초 간격 업데이트
- 에이전트 상태
- 실시간 로그
- 현재 시각

중간 빈도: 5초 간격 업데이트
- BOARD 요약
- 에러 카운트

저빈도: 30초 간격 업데이트
- 시스템 리소스
- 장기 통계
```

### 메모리 관리
```
로그: 최대 1000개 라인 메모리 유지
히스토리: 24시간치 요약 통계만 보관
캐시: 자주 조회되는 BOARD 상태 캐싱
```
