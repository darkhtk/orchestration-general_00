## 색상 및 상태 표시 가이드

### 상태 색상
- 🟢 **활성/정상**: 에이전트 실행 중, 정상 상태
- 🔴 **에러/중지**: 에러 발생, 에이전트 정지
- 🟡 **경고/대기**: 경고 상태, 대기 중
- ⚫ **비활성/정지**: 비활성화된 상태

### 우선순위 표시
- **P0** 🔴: 긴급 (즉시 처리)
- **P1** 🟠: 높음 (24시간 내)
- **P2** 🟡: 중간 (일주일 내)
- **P3** 🟢: 낮음 (여유있게)

### 프로그레스 바
```
진행률: [████████░░] 80%
대기열: [▃▅▇▆▄▅▇] 처리량 표시
시간: [●●●○○] 5단계 표시
```

## 키보드 단축키

### 전역 단축키
- `Tab` / `Shift+Tab`: 섹션 간 탐색
- `F1`: 도움말 표시
- `Ctrl+R`: 새로고침
- `Ctrl+C`: 프로그램 종료
- `Esc`: 메인 화면으로

### 섹션별 단축키
- `1-9`: 빠른 액션 실행
- `Enter`: 항목 상세보기
- `Space`: 일시정지/재개
- `/`: 검색 모드
- `f`: 필터 토글

## 반응형 레이아웃

### 최소 화면 (80x24)
- 에이전트 상태와 BOARD 요약만 표시
- 로그는 하단 2줄로 축소
- 상세 정보는 별도 모드로 분리

### 중간 화면 (120x30)
- 기본 4분할 레이아웃
- 모든 주요 정보 표시
- 스크롤 가능한 로그 영역

### 큰 화면 (160x40+)
- 추가 통계 패널 표시
- 더 많은 로그 라인
- 그래프와 차트 영역 확장
