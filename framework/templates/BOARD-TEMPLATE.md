| 이동 | 담당 | 비고 |
|------|------|------|
| 📋 Backlog → 🔧 In Progress | Developer | 자가 배정 허용 |
| 🔧 In Progress → 👀 In Review | Developer | SPEC 참조 필수 |
| 👀 In Review 결과 컬럼 기록 | Client | APPROVE / NEEDS_WORK |
| 👀 In Review → ✅ Done | Coordinator | APPROVE 확인 후 |
| 👀 In Review → ❌ Rejected | Coordinator | NEEDS_WORK 확인 후 |
| ❌ Rejected → 🔧 In Progress | Developer | 즉시 수정 착수 |
| 로드맵 상태 이모지 갱신 | Coordinator | Done/Rejected 이동 시 동시 |
